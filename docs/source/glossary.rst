
Glossary
========

.. glossary::
  
  cli
  
    Command Line Interface

  GitFlow
  
    GitFlow is a branching model for Git, created by Vincent Driessen. 
    It has attracted a lot of attention because it is very well suited to collaboration 
    and scaling the development team. 
    More on `GitFlow <https://datasift.github.io/gitflow/IntroducingGitFlow.html>`_.
    
  IDE
  
    Integrated Development Environment

  rst 
   
    reStructuredText is the default plaintext markup language used by Sphinx. A tutorial 
    can be found on `the Sphinx website <https://www.sphinx-doc.org/en/master/usage/restructuredtext/basics.html>`_
